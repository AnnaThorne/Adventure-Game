Objective

Your objective is to reach the final level of the game - CarrotFarm, to do that you'll have to traverse through the game world and interact with characters and items

Commands

  • go - command go moves player to the selected room
  • help - prints out a list of all available commands
  • exit - exits game
  • bag - shows currently held items
  • roomInfo - prints out information about the room you're currently in
  • open - attempts to open an object
  • take - attempts to take an object
  • talk - attempts to talk with a character
  • attack - attacks a character
